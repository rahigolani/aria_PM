**Live Site:** [View ARIA](https://<your-username>.github.io/aria-pm/)

## Quick Deploy

```bash
git init
git add .
git commit -m "Deploy ARIA site"
git branch -M main
git remote add origin https://github.com/<your-username>/aria-pm.git
git push -u origin main
```

Then enable GitHub Pages in **Settings → Pages → Source: main branch**.
